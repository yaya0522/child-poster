# 儿童识字小报生成器

一个基于 Nano Banana Pro API 的 HTML 应用，专为5-9岁儿童创建有趣的识字学习小报。

## 功能特点

- 🎨 **智能生成**：输入主题和标题，自动生成精美的识字小报
- 📚 **丰富词汇**：内置8个常见场景词汇库，每个场景包含15-20个核心词汇
- 🌈 **儿童友好**：色彩鲜艳、字体圆润、界面简洁
- 💾 **本地保存**：自动保存生成历史，支持图片下载
- 📱 **响应式**：支持手机、平板、电脑等多种设备
- 🔒 **安全存储**：API密钥本地加密存储

## 使用方法

### 1. 获取API密钥

1. 访问 [API Key管理页面](https://kie.ai/api-key)
2. 注册/登录账户
3. 获取您的 Nano Banana Pro API 密钥

### 2. 使用生成器

1. **输入API密钥**：首次使用时输入您的API密钥
2. **选择主题**：从下拉菜单选择预设主题（超市、医院、公园等）或输入自定义主题
3. **输入标题**：为小报起一个有趣的标题，如《走进超市》
4. **点击生成**：等待AI生成精美的识字小报
5. **下载保存**：生成完成后下载图片到本地

### 3. 快捷键

- `Ctrl/Cmd + Enter`：快速生成
- `Ctrl/Cmd + S`：下载当前图片
- `Ctrl/Cmd + N`：创建新的小报

## 技术特性

### 前端技术栈
- **HTML5**：语义化标签，无障碍访问
- **CSS3**：响应式设计，动画效果，渐变背景
- **Vanilla JavaScript**：模块化开发，ES6+语法

### 核心模块
- **prompt-generator.js**：智能提示词生成，场景词汇管理
- **api.js**：API调用封装，错误处理，任务轮询
- **app.js**：主应用逻辑，用户交互，历史记录管理
- **utils.js**：工具函数库，本地存储，DOM操作

### API集成
- 使用 Nano Banana Pro 的图片生成API
- 支持任务创建和状态查询
- 自动轮询机制，实时更新进度
- 完善的错误处理和重试机制
